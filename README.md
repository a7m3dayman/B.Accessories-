# B.Accessories Backend

### ما الذي تم إضافته؟
- خادم Node.js باستخدام Express.
- واجهة برمجية `GET /api/site-data` لقراءة البيانات.
- واجهة برمجية `POST /api/site-data` لحفظ التعديلات.
- واجهة برمجية `POST /api/reset` لإعادة البيانات الافتراضية.
- يتم تقديم الملفات الثابتة (`index.html`, `admin.html`, إلخ`) من نفس الخادم.

### كيفية التشغيل
1. افتح Terminal في مجلد المشروع.
2. شغل:
   ```bash
   npm install
   npm start
   ```
3. افتح المتصفح على:
   ```
   http://localhost:3000
   ```

### ملاحظة
- يجب فتح الموقع عبر `http://localhost:3000` لكي يعمل backend بشكل صحيح.
- عند تشغيل الخادم، التعديلات من لوحة التحكم ستظهر لكل المستخدمين الذين يزورون نفس الموقع.
